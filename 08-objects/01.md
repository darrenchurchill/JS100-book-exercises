# Launch School Introduction to Programming with JavaScript

## Ch. 8: Objects

### Exercise 1

Given the following code, how can you access the name of the person?
```js
let person = {
  name:       'Bob',
  occupation: 'web developer',
  hobbies:    'painting',
};
```

You can access the name of the person using:
```js
person.name;
```
or
```js
person['name'];
```
